Background

The Content Partner Sandbox is a consistent and reliable environment for testing applications built on top of EverFi's Content Partner SDK. It provides developers the ability to send and receive data between their application and our backend system, simulating how the application will perform once it is integrated into EverFi's production environment. Additionally, the Sandbox includes tools for monitoring application performance and visualizing application data.

Initial Setup

In order to begin testing your application in the Sandbox, your app must abide by the following rules.

1. Module JSON

Apps must be broken down into Modules, Activities, and Pages. Pages have specific logic associated with them, while modules and activities are used primarily as a grouping mechanism and to track progress. Users of an application will only direcly interact with pages. The SDK requires JSON representations of Modules, Activities, and Pages.
